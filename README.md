## Commit Rule

| Tag         | Description                           |
| ----------- | ------------------------------------- |
| ✨ Feat     | 기능구현 / UI 추가 등                 |
| 🙈 Chore    | 오타 수정 등, 일상 업무               |
| 🚚 Rename   | 변수명 변경(API에서 변경시 확인 필요) |
| 📝 Docs     | JSDoc 주석 추가 / README.md 변경 등   |
| ♻️ Refactor | 구조체 변경                           |
| 🛠️ Fix      | 버그 수정                             |
| 🎨 Style    | 스타일 수정                           |
